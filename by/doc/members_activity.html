Отслеживание текущей игровой активности участников клана

Отслеживается текущая активность участников клана за два периода - последние 7 и 30 дней. Общее кол-во боев и кол-во боев на ГК каждого игрока, а также количество боев по моделям танков (динамическая статистика). Статистика начинает накапливаться с момента добавления клана в базу сервиса.

Для каждого участника клана доступна страница с детальными данными о его игровой активности.

Премиум-режим обслуживания включает данные об игровой активности участников клана по моделям танков, расшифровку цифры боев за каждый день по моделям танков, историю боев на определенном танке по дням.
